# NetToolSample
### XmlSerialize
測試xml序列化，可以格式化、移除標頭或namespace
### XmlSigned
測試xml簽章，範例內有用pkcs7跟pfx取得X509憑證的作法
也有用官方文件RSA直接取得Key的作法
主要是要把原本的xml文件加簽，提升安全性
### CommandLine
在程式內使用終端機指令下語法

### StringExtension
- 字串轉全形或半形
- 判斷特殊字 大於ASCII table編碼127 的會無法轉換
### GenerateExcel
依照model產生Excel報表

### CreateFile
產生csv檔案寫入指定資料夾
